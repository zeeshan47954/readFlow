readFlow
A simple Android reading helper for people who lose focus on long text.

───

What it does

Helps you read long articles more easily by showing text slowly, one piece at a time, with tools to stay focused.

───

Main features

Read Article
• Paste your own text and read it in the app
• Or search the Stanford Encyclopedia and read scraped articles

Reader tools
• Text appears word-by-word / line-by-line
• Pause, play, go back, go forward
• Change speed, text size, and color
• Read aloud (text-to-speech)
• Tap a word to see its meaning
• Short AI summary of the article

Challenges
• 5 reading levels with harder text each time
• Same reader tools on every level

Focus Audio
• 3 ADHD focus audio tracks
• Play, pause, and resume where you left off

Get Help
• Links to ADHD/mental health sites (Talkspace, BetterHelp, SAMHSA, etc.)

───

Who it's for

People who struggle to read long or dense text without losing focus — especially some readers with ADHD.

It is not a full ADHD app. It does not track medication, habits, or daily life.

───

Tech used

• Java (Android)
• Firebase (database + audio storage)
• Text-to-speech
• Dictionary API (free, no key)
• OpenNLP for summaries

───

Setup

1. Open project in Android Studio
2. Add your google-services.json to the app/ folder
3. Enable Firebase Database and Storage
4. Upload adhd1.mp3, adhd2.mp3, adhd3.mp3 to Firebase Storage
5. Run on Android 7.0+ (API 24+)

───

App name idea

ADHDApp is too generic. Better names:

• ReadFlow — Read at your pace
• SteadyRead — One line at a time

───

Note

This app helps with reading focus, not all ADHD problems. Many people with ADHD may not need this — it’s built for those who do struggle with long text.
